# OpenInfiniFactory Goal

目标是复刻一个叫做 InfiniFactory 的 3D 方块解谜游戏。它通过回合制运算实现传送带运输、活塞推动等功能。

复刻版希望做成全平台版本，使用 Rust 的 Bevy 构建。

最初的一版目标：

- 在 macOS 上能直接运行，并呈现一个可见的 3D 版本。
- 支持打开背包和物品栏。
- 支持类似《我的世界》创造模式的放置和破坏方块。
- 放置方块不消耗物品，破坏方块不产生掉落物。
- 需要一个简单存档系统。
- 部分方块需要有四个朝向，并在放置时保留朝向。
- 如果需要图片素材，可以写简单 Python 脚本，通过 PIL 绘制纯色或色块素材，能表达含义即可。
