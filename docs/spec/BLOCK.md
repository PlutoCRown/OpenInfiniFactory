# Block Specification

游戏中的方块分为三类：

- **工厂方块**：玩家放置的、有逻辑功能的方块。
- **场景方块**：场景中的无功能方块。
- **材料方块**：由材料产生器生成的方块。

材料方块在逻辑中可以组合成一个“结构”。单个材料方块也视为一个结构。

## 第一阶段方块

优先完成下面三个方块功能，再开始构建后续方块。

### 材料产生器

- 放置后可配置材料产出的频率。
- 放置后可配置产出的方块类型。
- 产生的材料方块具有重力效果。
- 每个回合，材料方块下落一格。

### 焊接器

- 有四个方向。
- 放置后会在自身前方生成一个焊点方块。
- 焊点方块没有碰撞。
- 如果焊点位置被其他方块占用，则该焊接器不生效。
- 如果两个焊点方块相邻，它们应显示为连接样式。
- 每个回合，如果两个结构同时存在于相连位置，并且它们都有焊点方块，则两个结构焊接为一个整体结构。
- 单个方块也可以被焊接为结构。

### 传送带

- 有四个方向。
- 每个回合检查自身上方是否有可移动的结构。
- 如果有结构，检查该结构是否可以朝传送带方向移动。
- 如果可以移动，则移动该结构。

## 第二阶段方块

### 方块识别器

- 有四个方向。
- 检查前方是否出现材料方块。
- 如果检测到材料方块，则其他五个面输出信号。

### 电线

- 自动连接附近的其他电线。
- 连接活塞和方块识别器的可输出信号面。
- 可以传递信号。

### 活塞

- 正面可以推出。
- 其他五个面可以连接电线。
- 通电时推出前方结构。
- 推出前需要检查移动合法性；如果不合法，则不会推出。
