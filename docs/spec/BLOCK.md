# Block Specification

游戏中的方块分为三类：

- **场景方块**：场景中的无功能方块，只能在编辑期放置。
- **工厂方块**：玩家放置的、有逻辑功能的方块，只能在游玩期放置。
- **材料方块**：由材料产生器生成的方块，玩家不能直接放置。

模拟期中，工厂方块和材料方块都会受到重力影响。场景方块不会下落。

场景方块和工厂方块会自动和周边方块相连。只有没有连接到任何场景方块的悬空工厂方块结构才会下落。

材料方块在逻辑中可以组合成一个“结构”。单个材料方块也视为一个结构。材料方块只能通过焊接器和其他材料方块焊接，不能与场景方块或工厂方块焊接。

## 第一阶段方块

优先完成下面三个方块功能，再开始构建后续方块。

### 材料产生器

- 放置后可配置材料产出的频率。
- 放置后可配置产出的方块类型。
- 产生的材料方块具有重力效果。
- 每个回合，材料方块下落一格。

### 焊接器

- 有四个方向。
- 放置后会在自身前方生成一个焊点方块。
- 焊点方块没有碰撞。
- 如果焊点位置被其他方块占用，则该焊接器不生效。
- 如果两个焊点方块相邻，它们应显示为连接样式。
- 每个回合，如果两个结构同时存在于相连位置，并且它们都有焊点方块，则两个结构焊接为一个整体结构。
- 单个方块也可以被焊接为结构。

### 传送带

- 有四个方向。
- 每个回合检查自身上方是否有可移动的结构。
- 如果有结构，检查该结构是否可以朝传送带方向移动。
- 如果可以移动，则移动该结构。

## 第二阶段方块

### 方块识别器

- 有四个方向。
- 检查前方是否出现材料方块。
- 如果检测到材料方块，则其他五个面输出信号。

### 电线

- 自动连接附近的其他电线。
- 连接活塞和方块识别器的可输出信号面。
- 可以传递信号。

### 活塞

- 正面可以推出。
- 其他五个面可以连接电线。
- 通电时推出前方结构。
- 推出前需要检查移动合法性；如果不合法，则不会推出。

## 第三阶段方块

### 抬升器

- 检查自身上方五格内是否有结构。
- 如果有结构，则每回合将该结构向上抬升一格。
- 持续抬升该结构，直到它移出抬升器上方五格范围之外。
- 抬升前需要检查移动合法性；如果不合法，则不会抬升。

### 旋转器

- 每回合检查自身上方是否有结构。
- 如果有结构，则将该结构绕垂直轴旋转 90 度。
- 每回合只旋转一次。
- 旋转前需要检查目标位置是否合法；如果不合法，则不会旋转。

### 阻拦器

- 有四个方向。
- 行为类似活塞，但信号逻辑相反。
- 其他五个面可以连接电线。
- 没有信号时，阻拦器向正面伸出并阻挡结构。
- 收到信号时，阻拦器收回。

### 钻头

- 有四个方向。
- 会在自身正面生成一个钻头头方块。
- 钻头头方块接触到结构时，会破坏该结构中的材料方块。
- 如果被破坏的材料方块是结构中的唯一连接，结构可能被断成两个或多个结构。

### 激光

- 有四个方向。
- 其他五个面可以连接电线。
- 当其他五个面接收到电力信号时，从正面发射激光。
- 激光会检查正前方 30 格。
- 激光只能破坏材料方块。
- 如果激光遇到工厂方块或场景方块，则被阻拦。
