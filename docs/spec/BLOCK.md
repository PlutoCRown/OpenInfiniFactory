游戏中分为工厂方块、场景方块和材料方块。
工厂方块是玩家放置的有逻辑的方块。
场景方块为场景中的无功能方块。
材料方块为材料产生器里产生的方块。

材料方块和材料方块在下面的逻辑中可能会被形成一个结构，单个材料方块也是一个结构。

第一个是材料产生器，它放下来之后，可以配置材料产出的频率和产出的方块。
材料方块具有重力效果，每个回合下落一格。

第二个是焊接器，它有四个方向。当它放下来之后，会在它的前方生成一个焊点方块。这是一个没有碰撞的方块，如果该位置被其他方块占掉，那么这个焊接器就不会生效。如果两个焊点方块相邻，那么它们就会有连接在一起的样式。然后每一回合如果有两个结构同时存在于相连的位置，并且它们都有焊点方块的话，那么它们两个结构就会被焊在一起。形成一个整体的结构。（注意单个方块也可以是）


第三个是传送带，传送带有四个方向，它的功能是在每回合中，检查它上面有没有可移动的【结构】如果有的话，检查这个结构是否可以往对应的方向移动？如果可以，那么移动。
