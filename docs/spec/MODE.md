# Mode Specification

游戏玩家分为两类操作模式：

- **编辑模式**
- **游玩模式**

所有方块定义见 [Block Specification](BLOCK.md)。

## 编辑模式

- 可对场景进行编辑操作。
- 可放置多个场景方块。

## 游玩模式

- 可放置工厂方块。
- 玩家可以在暂停菜单中快速切换编辑模式和游玩模式。

## 回合制运行

游玩模式下，游戏以回合制运行。

- 默认每 1 秒运行 1 回合。
- 默认进入游玩模式时处于暂停状态。
- 右上角 UI 显示回合运行状态。
- UI 至少包含暂停、播放、加速和回滚等功能。

### 播放

- 当玩家切换到播放状态时，材料产生器开始产生材料方块。
- 玩家可以随时暂停。
- 玩家可以加速播放。

### 回滚

- 回滚到刚开始播放时的状态。
- 清除场上所有材料方块及其形成的结构。
- 工厂方块回归播放开始时的初始状态。
